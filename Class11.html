Glossary of Web Development Terms

HTML
HyperText Markup Language, used to create a web page.
CSS
Cascading Style Sheets, used top style or format the layoput of your web page.
JavaScript
A programing language used to create interactive and dynamic web content.
Responsive Design
A design approach ensuring web pages look good on all devices, from desktop to smartphone.
API
Application Programing Interface, a set of tools and protocls for building softwear and application.
